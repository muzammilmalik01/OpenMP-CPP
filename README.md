
## Installation on Local Machine
NOTE: This is for Linux Machine only.

Install OpenMP
```bash
$ sudo apt-get install libomp-dev
```
Git Clone

```bash
$ git clone https://github.com/muzammilmalik01/OpenMP-CPP.git

```
Compiling 
```bash
$ g++ -fopenmp your_code.cpp -o your_executable 
```
Executing
```bash
$ ./your_executable
```
